# Базовые команды git

## Проверка работоспособности и версии git

```bash
git --version
```

## Настройка git с помощью настройки имени и почты

```bash
git config --global user.name "My Name"
git config --global user.email "myemail@example.com"
```

## Проверка настроек 

```bash
git config --list
```
